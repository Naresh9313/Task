# SellSharp Landing Page

A modern, responsive landing page built using React + TypeScript for SellSharp.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Features
- Fully component-based architecture
- Mobile, tablet & desktop responsive
- Clean and maintainable TypeScript code
- Modern UI/UX

## Setup
npm install
npm run dev
